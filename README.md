# Loan Risk Analysis

## Overview
This project analyzes loan applicant data to assess risk and identify factors affecting loan default.

## Tools Used
- Excel
- Power BI

## Dataset
- Financial and demographic data of applicants
- Includes income, credit history, loan amount

## Work Done
- Data cleaning and preprocessing
- Risk segmentation
- Dashboard creation

## Key Insights
- Low credit score = higher default risk
- High loan-to-income ratio increases risk
- Stable employment reduces default probability
